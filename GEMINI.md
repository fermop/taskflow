# Reglas del Proyecto: Gestor de Tareas
- **Arquitectura:** Screaming Architecture. La lógica de negocio debe vivir en `src/features/[feature]/services`.
- **Stack:** Next.js (App Router), Tailwind CSS, Firebase Firestore.
- **UI:** Usamos componentes de Shadcn/UI (ubicados en `src/components/ui`).
- **Commits:** Seguir Conventional Commits estrictamente.
- **Firebase:** Usar la instancia configurada en `src/lib/firebase.ts`.
